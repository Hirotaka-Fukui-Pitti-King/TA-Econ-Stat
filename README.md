# Clean-JPSED

**Code for transforming raw [Japanese Panel Study of Employment Dynamics (JPSED) ](https://www.works-i.com/english/reports/jpsed.html) data into analysis-ready data for empirical research.**

This repository contains code to clean, harmonize, and structure raw survey data from the Japanese Panel Study of Employment Dynamics (JPSED). It is designed to construct consistent panel datasets suitable for empirical analysis while improving transparency and reproducibility in data preparation.

# Clean-JPSED

**Code for transforming raw JPSED data into analysis-ready data for reproducible research.**

This repository contains code to clean, harmonize, and structure raw survey data from the [Japanese Panel Study of Employment Dynamics (JPSED) ](https://www.works-i.com/english/reports/jpsed.html). It is designed to construct consistent panel datasets suitable for empirical analysis while improving transparency and reproducibility in data preparation.

## Prerequisites 

Access to the JPSED data requires registration with **SSJDA Direct** and the acquisition of a user ID and password. For further information, see the [SSJDA Direct User’s Guide](https://csrda.iss.u-tokyo.ac.jp/english/infrastructure/access/).

# Clean-JPSED

**R code for transforming raw JPSED data into analysis-ready data for reproducible research.**

This repository contains R code to clean, harmonize, and structure raw survey data from the [Japanese Panel Study of Employment Dynamics (JPSED) ](https://www.works-i.com/english/reports/jpsed.html). It is designed to construct consistent panel datasets suitable for empirical analysis while improving transparency and reproducibility in data preparation.

## Prerequisites 

Access to the JPSED data requires registration with SSJDA Direct and the acquisition of a user ID and password. For further information, see the [SSJDA Direct User’s Guide](https://csrda.iss.u-tokyo.ac.jp/english/infrastructure/access/).

## How to use

1. Install R
   - it is recommend to use [rig](https://github.com/r-lib/rig) to install R
1. Clone this repository (or download it as a ZIP file)
